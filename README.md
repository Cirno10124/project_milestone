# Project Milestone（项目总览）

本仓库包含 **后端（NestJS + TypeORM + MySQL）** 与 **前端（Vue 3 + Vite + Pinia）**，用于项目计划/任务/WBS/关键路径分析，并支持与 GitLab 联动（提交时自动更新任务进度）。

---

- **前端运行说明**：[`frontend/project_milestone_frontend/README.md`](frontend/project_milestone_frontend/README.md)
- **项目管理使用说明（组织/项目/WBS/任务/关键路径/甘特）**：[`docs/project-management.md`](docs/project-management.md)

---

## 仓库结构

- `backend/`：后端服务（API、鉴权、多租户组织、项目/任务/WBS、迁移等）
- `frontend/project_milestone_frontend/`：前端应用（页面、状态管理、接口封装等）
- `docs/`：项目文档（如 Git 联动说明）
